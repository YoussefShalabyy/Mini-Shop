# Mini Shop - Web Dashboard

Built with **Vite**, **React 18**, **TypeScript**, and **Zustand**.

## Overview
A responsive administrative portal to manage the Mini Shop e-commerce platform.

### Features
- **KPI Dashboards:** Real-time revenue and status distribution charts using Recharts.
- **Order Pipeline:** Update order statuses directly from the data tables.
- **Product Management:** Full CRUD capabilities for the shop catalogue.
- **Design:** Custom Vanilla CSS design system replacing standard Tailwind for a more premium, glassmorphism aesthetic.
- **State:** TanStack React Query for efficient data fetching and caching.

## Setup
1. Copy `.env.example` to `.env`.
2. Run `npm install`.
3. Run `npm run dev`.

The dashboard will open on `http://localhost:5173`.

*Note: You must log in with an account that has the `admin` role.*
